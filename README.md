# New Computer Checklist
A checklist and list of scripts I should run when getting a new Computer.

# Ideal dev computer (as of 2016)
- iMac
- Quad-core >=4.0GHz Processor
- 32GB DDR3
- 512GB Flash Storage
- 4GB Video Memory
